Disaster Category Dataset
==========================

75 images per category, selected as an even spread across each source
folder for visual variety, then compressed to JPEG.

  Earthquake  -> 75 images
  Fire        -> 75 images
  Flood       -> 75 images
  Normal      -> 75 images

Total size: ~3.7 MB (well under the 25 MB budget)
